Bulk Import Channels

Select Plex/Jellyfin playlists & collections, TV shows, and custom programming (same content resources as channel Properties). If that channel already exists, its programming is updated (duplicates removed, order randomized) instead of creating a new channel. Content sources are saved on the channel for later re-sync.

Leave empty to name each channel after its content source. If set, that name is used (all selected sources import into that one channel). For Letterboxd/Trakt tracked lists, use Library → Lists.
Configure a Plex or Jellyfin server in Settings, or create custom shows under Library first.
