Bulk Import Channels

Select Plex playlists, collections, and/or TV shows. Each item maps to a channel with the same name. If that channel already exists, its programming is updated (duplicates removed, order randomized) instead of creating a new channel. Content sources are saved on the channel for later re-sync. If multiple sources share a name, both are imported into that one channel.

Configure a Plex server in Settings first.
