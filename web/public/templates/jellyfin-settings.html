Stream path settings

Choose how dizqueTV opens Jellyfin media. Network streams over HTTP from Jellyfin (works remotely). Direct reads files from disk on this machine (faster seeks; requires shared storage and optional path replacement).

Path replacements

Jellyfin reports file paths as seen on the Jellyfin host. If this machine uses a different mount (e.g. /media/movies → Z:\movies), set the original prefix and the local replacement.

Direct mode uses the media Path from Jellyfin (after replacement). If the file is not readable here, playback falls back to the network stream. Re-sync libraries after changing paths if programs were cached without file paths.

Network path selected — media is streamed from the Jellyfin server over HTTP. Switch to Direct to use local files and path replacement.
