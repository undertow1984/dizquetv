Library

Components that will allow you to organize your media library to help with the creation of channels that do things the way you want.

Filler...

Filler lists are collections of videos that you may want to play during 'flex' time segments. Flex is time within a channel that does not have a program scheduled (Usually used for padding).

Custom Shows....

Custom Shows are sequences of videos that represent a episodes of a virtual TV show. When you add these shows to a channel, the schedule tools will treat the videos as if they belonged to a single TV show.

Management...

Enable libraries, sync Plex and Jellyfin content to a local cache, and see last full sync times. Cached data is used across the app so channel tools avoid repeated media-server API scans.

Lists...

Track Letterboxd and Trakt movie lists, match them to your library cache, and refresh anytime (similar to Radarr/Sonarr list sync). Optionally keep a channel updated from each list.

Library Management (Plex + Jellyfin)

Toggle libraries on or off, and sync movies / shows / collections / playlists into a local cache. After the first full sync, later syncs only process adds, updates, and deletes. Disabled libraries are hidden from programming tools. Mark Contains filler on libraries that should appear when adding filler clips.
